# 插入排序 #
- 直接插入排序
- ==Shell排序==
# 交换排序 #
- 冒泡排序
	- 依次向后交换
- ==快速排序==
	- 每次将一个元素放到对应位置
	- 每个分划区间中，以第一个元素为基准，两个指针从两侧向中间扫，发现不符合顺序的进行交换，最后当指针异位时将基准与右侧指针交换
	- 在全部倒序的最坏情况下会退化为$O(n^2)$
# 选择排序 #
- 直接选择排序
- ==堆排序==
	- 堆按层序进行初始化为大根堆
	- 先选择当前未被排序的最大数移到对应位置，再令堆顶下沉，重建为大根堆，如此循环

- ==归并排序==
	- 递归从大到小归并，非递归从小到大归并 *（选填）*
# 稳定性 #
- 相同元素的相对位置不变
- ==不稳定的排序==：
	- 选择排序（直接选择排序&堆排序）
	- 快速排序
	- Shell排序
