# 基本概念 #
- 联通图：任意两个不同的顶点都可及的无向图
- ==n个顶点无向连通图，至少n-1条边==
- ==n个顶点有向连通图，至少n条边==
# 存储结构 #
- 邻接矩阵
- 邻接表
# 遍历算法 #
- DFS
	- 循环+递归
	- 栈（非递归）
- BFS
	- 队列
# 拓扑排序 #
- 入度为0时加入序列
- 前驱节点加入序列时入度减一
- ==拓扑序列==
	- 找到关键活动
	- 确定顺序
	- 插入剩余活动
# 关键路径 #
- 关键路径：源点到汇点的具有最大长度的路径
- 关键活动：关键路径上的所有活动
	- 全部提前完成可使任务提前完成
	- 有一个推迟完成使任务推迟完成
# 最短路径 #
- Dijsktra
	- 每次更新进行比较，将距离原点路径最短的点加入集合
	- 更新：$min(l(v), l(u)+w)$
	- O(nlogn+e)
	- 不能处理负权图
