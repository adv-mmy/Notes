# 关键路径 #
- 关键路径：源点到汇点的具有最大长度的路径
- 关键活动：关键路径上的所有活动
	- 全部提前完成可使任务提前完成
	- 有一个推迟完成使任务推迟完成
# 最短路径 #
- Dijsktra
	- 源点到每个点的最短距离
	- 每次更新进行比较，将距离原点路径最短的点加入集合
	- 更新：$min(l[v], l[u]+w)$
	- $O(nlogn+e)$
	- 不能处理负权图
	- 贪心
- Floyd
	- 每个点间最短距离
	- 三层循环，每次更新每个点到目标点的距离
	- 更新：$min(l[i][j], l[i][k]+l[k][j])$
	- $O(n^3)$
	- 能处理负权边，不能处理负环
	- dp
# 最小支撑树 #
- Prim
	- 每次更新将最小跨边加入集合，将不属于集合的端点加入集合，直到不存在跨边 *（S=V）*
	- 初始选定一个源点，再在此基础上判断最小跨边
	- 稠密图
	- $O(n^2)$
- Kruskal
	- 在G中选择权值最小的边，并将此边从G中删除
	- 若该边加入T后不产生环（即此边的两个端点在T的不同连通分量中），则将此边加入T中，从而使T减少一个连通分量，否则本步骤无操作。
	- 重复至T中仅剩一个连通分量
	- 稀疏图
	- $O(eloge)$
