# 基本概念 #
- 度：一个结点儿子的个数
- 层数
	- 根的层数为0
	- 其余结点的层数为其父亲的层数加1
# 二叉树 #
- 遍历算法
	- 先根
	- 中根
	- 后根
	- 上述三个算法的非递归形式 *（栈）*
	- 层序 *（队列）*
- 哈夫曼树：
	- 每次选取输入序列中权值最小的两个结点，加和生成新结点，并将这个加和关系体现为二叉树的父子关系
	- 哈夫曼编码：哈夫曼树左枝赋0，右枝赋1，==从根到叶==的二进制串即为叶结点的哈夫曼编码
- 左儿子右兄弟 *（树转化为二叉树）*
	- 所有兄弟结点间连线
	- 只保留大儿子和大兄弟的连线
- 森林转化为二叉树
	- 每棵树都按左儿子右兄弟结构转化为二叉树
	- 后续树作为前一棵树的右子树
- Father链接结构
	- ”向上“访问能力
	- 便于查找指定结点的祖先
	- 并查集
- 左儿子右兄弟链接结构
