书P314~P316

- **基址寻址**：操作数的有效地址EA等于指令字中的形式地址与基址地址寄存器中的内容（*基地址*）相加
- **变址寻址**：操作数的有效地址EA等于指令字中的形式地址与变址地址寄存器IX的内容相加之和
- 都可有效地==扩大==指令寻址范围

|         | 基址寻址    | 变址寻址     |
| ------- | ------- | -------- |
| 基准地址来源  | 基址寄存器   | 形式地址A    |
| 偏移量来源   | 形式地址A   | 变址寄存器    |
| 寄存器内容设定 | 系统      | 用户       |
| 适用范围    | 程序动态重定位 | 数组或字符串处理 |
